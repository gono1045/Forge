# Forge（CLIゲーム）

## 概要

Forgeは、鉱石を収集しインゴットを生成、装備を強化しながら進行するCLIベースのシミュレーションRPGです。
経済要素（売買・価格変動）と戦闘を組み合わせた構成になっています。

---

## 動作環境

- macOS（動作確認済み）
- ターミナル環境が必要

※ 本アプリケーションはMac専用です。WindowsおよびLinuxでは動作しません。

---

## 起動方法（Mac）

1. zipを解凍
2. run.command をダブルクリック
3. ゲームが起動します

※ gameファイルは直接実行しないでください

---

## ゲームの流れ

1. 鉱石を購入
2. インゴットに精錬
3. 売却して資金を増やす
4. 武器・防具を強化
5. 冒険で素材と資金を獲得

---

## 主な機能

- 鉱石購入 / インゴット精錬 / 売却
- 武器・防具の強化
- ターン制バトル（攻撃・防御・アイテム・逃走）
- ショップ機能
- セーブ / ロード機能
- インゴット価格の変動

---

## セーブについて

- セーブデータは data/save.dat に保存されます
- ゲーム内メニューからセーブおよびロードが可能です

---

## 注意事項

- 初回起動時にセキュリティ警告が表示される場合があります
  その場合は右クリックから「開く」を選択してください
- run.command から起動してください
- ターミナル上で動作します
